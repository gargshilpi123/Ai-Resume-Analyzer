# Ai-Resume-Analyzer
AI Resume Analyzer is a web-based application built using Python and Streamlit that helps users analyze resumes intelligently. The system extracts key information from resumes, evaluates skills, predicts suitable job domains, and provides personalized recommendations to improve resume quality.
The application uses Natural Language Processing (NLP) techniques and the spaCy library to parse resume content efficiently. It also stores user data and feedback securely in a MySQL database, enabling structured analysis and future insights.

Key Features:

1. Upload and analyze resumes (PDF/DOCX)

2. Automatic skill extraction using NLP

3. Resume scoring and feedback generation

4. Job domain prediction based on resume content

5. Recommended skills and courses

6. User feedback collection

7. Interactive UI built with Streamlit

8. MySQL database integration

Tech Stack

1. Python

2. Streamlit

3. spaCy

4. PyResParser

5. MySQL

6. Pandas & NumPy

7. NLTK

Use Case:

This project is useful for:
1. Students and freshers improving their resumes

2. Job seekers wanting role-based recommendations

3. Recruiters for quick resume screening

4. Learning NLP-based document analysis
